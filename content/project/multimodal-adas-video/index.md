This project line studies how first-person traffic video, social media crash evidence, and ADAS-equipped vehicle events can support richer safety understanding.

Current work includes the SAVeD dataset for near-miss and crash event analysis, LLM-assisted event parsing, vision-language modeling, and controllable diffusion-based counterfactual scenarios for "what-if" safety assessment.
