This project line develops proactive crash risk and crash type prediction models using connected vehicle trajectories, traffic states, risky-driving behavior, and video-derived context.

Representative methods include BiLSTM-Transformer sequence models, deep ensembles, causal feature interpretation, and macro-micro exposure fusion for behavior-aware freeway safety assessment.
